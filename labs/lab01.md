# Lab 1

## Objectif du TP

Ce TP vise à mettre en place l'environnement de travail nécessaire à la réalisation des TP et à implémenter en utilisant GitHub Copilot l'application qui sera utilisée pour les différents TPs.

## Pré-requis

1. Installer [git](https://git-scm.com/book/fr/v2/D%C3%A9marrage-rapide-Installation-de-Git)
2. Installer [Git Credential Manager](https://github.com/git-ecosystem/git-credential-manager/blob/release/docs/install.md)
3. Installer [vscode](https://code.visualstudio.com/)
4. Installer l'extension vscode [Vue (Official)](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
5. Installer [pnpm](https://pnpm.io/installation)
6. Installer la dernière version stable de Node.js (LTS)
    - si pas déjà installée sur la machine
    - en utilisant pnpm via la commande `pnpm env use --global lts`
7. S'assurer que GitHub Copilot ([version Pro via compte étudiant](https://education.github.com/pack)) est activé dans vscode

## Instructions

Implémenter l'application 2048 en utilisant GitHub Copilot pour générer le code. Le code de l'application doit être écrit en Nuxt avec TypeScript et Tailwind CSS.

## Livrables

1. Un dépôt GitHub contenant le code de l'application 2048
2. Un fichier de notes indiquant ce que vous avez appris sur ce TP, la méthode adoptée pour arriver à réaliser cette application et les difficultés rencontrées.
